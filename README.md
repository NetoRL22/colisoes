* [setup.jsonc](https://github.com/NetoRL22/colisoes/blob/main/setup.jsonc) é o arquivo de configuração da simulação, altera os parametros do programa.
* [loop.exe](https://github.com/NetoRL22/colisoes/blob/main/loop.exe) é o arquivo executável do programa, para executar sua simulação faça:  
`.\loop.exe setup.jsonc`  
no cmd/terminal da pasta onde esses arquivos se encontram
* outros arquivos são o código-fonte do programa, loop é o pai, ga é um arquivo meu que pode ser encontrado em um repositório externo e render e fisica arquivos com a lógica mas pesada do programa
